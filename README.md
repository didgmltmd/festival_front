실시간 축제 주문·조리·서빙 현황을 한눈에 보여주는 프론트엔드입니다. A/B/C 조리 구역과 카운터(계산/픽업) 화면을 분리해 운영 효율을 극대화하고, WebSocket으로 주문 상태를 즉시 동기화합니다.

핵심 기능 (Features)

실시간 주문 동기화: Socket.IO 기반으로 신규 주문, 조리 시작/완료, 서빙 완료를 즉시 반영

구역 분리 운영: A/B/C 구역별 대시보드(대기→조리중→완료 파이프라인)

카운터 화면: 픽업번호 호출/완료 처리, 품목별 매출/수량 집계(가벼운 실시간 통계)

알림/효율 UI: 시각적 우선순위, 조리 지연 하이라이트, 키보드 단축키(선택)

오프라인 내구성(선택): 일시 연결 끊김 시 로컬 큐에 작업을 임시 저장 후 재전송



아키텍처 개요

프레임워크: React (권장: React 19)

번들러: Vite

UI: MUI(또는 Tailwind + Radix), 아이콘: lucide-react

상태관리: Zustand (혹은 React Query + Context 조합)

실시간: socket.io-client
